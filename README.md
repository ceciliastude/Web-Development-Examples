A Resident Evil inspired website done during my studies alongside 2 other collaboration partners.


Software used: VSC, Microsoft Databases
Languages: HTML, DHTML, XML, CSS, JavaScript
Frameworks: Node.js, JSON, Express, Ajax


HOW TO USE:

1. Clone this repo and open the file in a coding software like Visúal Studio Code or Visual Studio
2. Open a terminal and type "node index.js". This will keep the server running and will only stop when you end the run.
3. Open a window on the web and type "Localhost:3000".
4. Enjoy scrolling around the website! For login information, type (IT21 OR IT22)-(00-18) as username. For example "IT22-08". Then type tricell as the password.
